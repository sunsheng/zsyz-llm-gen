## 七、关键发现

1. **页面代码为主**：`views/` 占总行数 75.7%，页面组件是项目主体
2. **Vue 单文件占绝对多数**：`.vue` 文件行数占比 84.4%，符合 SPA 项目特征
3. **精准招商模块体量最大**：39 个页面、5,233 行，占总页面数 30%
4. **Mock 数据层较重**：`api/mock/` 1,651 行，占 api 层 78%，为全量 Mock 阶段
5. **基础设施精简**：`composables` + `stores` + `utils` 合计仅 639 行（1.9%），复用度高
