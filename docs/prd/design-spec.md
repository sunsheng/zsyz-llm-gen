## 二、UI 风格说明

### 2.1 设计规范

基于 Playwright 截取的两个参考系统登录页分析：

| 属性           | 值                                                               | 说明                               |

| **深蓝强调**   | `#0970F0`                                                        | hover/active 状态                  |

| **次要文字**   | `#909399`                                                        | 辅助/提示文字                      |
| **边框色**     | `#DCDFE6`                                                        | 分割线/边框                        |
| **成功**       | `#67C23A`                                                        | 正向指标/上涨                      |

### 2.4 组件风格公约

#### 卡片样式

```scss
.content-card {
  background: #ffffff;
  border-radius: 4px;
  box-shadow: 0 2px 12px 0 rgba(0, 0, 0, 0.06);
  padding: 20px;
  margin-bottom: 16px;
}
```

#### 统计卡

```
+----------------------------------+
| [📊 Icon 32px]   招商项目总数     |
|                  1,286           |
|           ↑ 12.5%  vs 上月        |
+----------------------------------+
```

- 图标颜色: 主蓝色 或按指标类别着色
- 数值: 28-32px 粗体, #303133
- 趋势箭头: 绿色↑(正向) / 红色↓(负向)

#### 数据表格

- 斑马纹 (`stripe`)
- 边框 (`border`)
- 表头背景: `#FAFAFA`, 字重 600
- 行悬停: `cursor: pointer`, 背景 `#F5F7FA`
- 尺寸: `default` (中等)

#### 滚动条

```css
::-webkit-scrollbar {
  width: 8px;
  height: 8px;
  border-radius: 4px;
}
::-webkit-scrollbar-thumb {
  border-radius: 4px;
  background: rgba(0, 0, 0, 0.1);
}
```
