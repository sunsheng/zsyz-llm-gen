## 四、页面布局与组件结构说明

### 4.1 六种页面类型分类

根据功能特性，将全部 ~140 个页面归纳为以下 **6 种页面模板**：

---

#### **Type A: 仪表盘 / 分析页** (约 35 页)

**适用模块**: 运行分析(大部分)、企业监测(部分)、区域对标(部分)、风险预警(部分)

**典型页面示例**: 产业招商数据概览、盈利能力、产业规模及预测、关键指标看板、五维评价模型

**组件结构**:

```
PageContainer
├── PageHeader                          // 页面标题 + 面包屑 + 操作按钮组
│
├── StatCardsRow                        // KPI 统计卡行 (flex 横排 4 列)
│   ├── StatCard                        // 单个统计卡 (icon + label + value + trend)
│   ├── StatCard
│   ├── StatCard
│   └── StatCard
│
├── TabBar (可选)                       // 子页面 Tab 切换栏
│
├── ChartSection                        // 图表区域 (grid 网格布局)
│   ├── ChartPanel                      // 单个图表容器
│   │   └── LineChart / BarChart / PieChart / RadarChart ...
│   ├── ChartPanel
│   ├── ChartPanel
│   └── ChartPanel
│
└── TableSection                        // 数据表格区域
    ├── FilterBar                       // 搜索 + 筛选下拉框
    └── DataTable                       // el-table (排序/分页)
```

**布局示意**:

```
+------------------------------------------------------------------+
|  产业招商数据概览                              [导出] [刷新]     |
+------------------------------------------------------------------+
| +------------+ +------------+ +------------+ +------------+       |

| +------------+ +------------+ +------------+ +------------+       |
+------------------------------------------------------------------+
| [数据概览] [空间数据] [渠道数据] [管理数据]                            |
+------------------------------------------------------------------+
| +-----------------------+ +-----------------------+               |
| |  入驻率趋势 (折线图)    | |  热门房源排行 (柱状图) |               |
| |                       | |                       |               |
| +-----------------------+ +-----------------------+               |
| +-----------------------+ +-----------------------+               |
| |  租售状态分布 (饼图)    | |  各园区对比 (雷达图)    |               |
| |                       | |                       |               |

+------------------------------------------------------------------+
|  详细数据列表                                                        |

#### **Type B: 地图页** (约 15 页)

**适用模块**: 产业地图(全部9页)、区域对标(热力图)、区域地图(部分)

**典型页面示例**: 产业资源搜索、产业空间分布、规上企业分布、地图热力图

**组件结构**:

```
PageContainer
├── PageHeader
│
├── MapLayout (flex 横向分割)
│   ├── MapControlPanel (左侧控制面板, ~360px)
│   │   ├── SearchBox                  // 搜索输入框
│   │   ├── FilterForm                 // 条件筛选表单
│   │   ├── LayerToggleGroup           // 图层开关组
│   │   ├── SearchResultList           // 搜索结果列表
│   │   └── LegendPanel                // 图例面板
│   │
│   └── MapContainer (flex-1 占满剩余)
│       ├── MaptalksMapWrapper         // Maptalks 地图画布
│       │   ├── VectorLayer            // 矢量图层 (边界/区域)
│       │   ├── MarkerLayer            // 标记图层 (企业/集群)
│       │   ├── ClusterLayer           // 聚合标记层
│       │   └── HeatmapLayer           // 热力图层 (百度叠加)
│       ├── MapToolbar                 // 工具条 (缩放/全屏/量测/复位)
│       ├── MapLegend                  // 右下角图例
│       └── PopupOverlay               // 点击弹窗信息浮层
```

**布局示意**:

```

| 🗂️[图层]|            |                   |                          |
|         |            |                   |  右下角: 图例            |

```

#### **Type C: 图谱 / 关系图页** (约 12 页)

**适用模块**: 产业图谱(8页)、关系链招商、企业画像(关系图谱)

**典型页面示例**: 产业链图谱构建、产业链关联企业、关系图谱

**组件结构**:

```
PageContainer
├── PageHeader
│   ├── IndustryChainSelect          // 产业链选择下拉
│   └── ActionButtons                // [导出] [编辑] [刷新]
│
├── GraphLayout
│   ├── GraphCanvas (全宽 D3/SVG 力导向图画布)
│   │   ├── NodeRenderer             // 节点渲染 (圆形, 半径∝重要性)
│   │   ├── EdgeRenderer             // 边渲染 (曲线连线 + 方向箭头)
│   │   ├── ZoomControls             // 缩放/平移控件
│   │   └── Minimap                  // 小地图导航
│   │
│   └── NodeDetailDrawer (右侧抽屉, ~400px, 可滑出)
│       ├── BasicInfoSection         // 基本信息
│       ├── RelationNetwork          // 子关系网络 (迷你图)
│       ├── EnterpriseList           // 该环节关联企业列表
│       └── ActionButtons            // [查看详情] [添加关注]
```

**布局示意**:

```

|         |         /   |                    | |                 | |
|    ─────○─────────○    |                    | | 所属环节: 上游   | |
|        / \          \___/                   | | 关联企业: 34家   | |
|       ○   ○                                 | |                 | |

#### **Type D: 画像 / 详情页** (约 18 页)

**适用模块**: 企业监测(画像/总览/健康度)、精准招商(评估/画像)

**典型页面示例**: 企业多维度画像、龙头企业分析、带动能力画像、资质企业需求画像

**组件结构**:

```
PageContainer
├── PageHeader (带返回按钮)
│
├── PortraitLayout (横向分割)
│   ├── LeftProfilePanel (固定宽度 ~400px)
│   │   ├── EntBasicInfoCard          // 企业基本信息卡片
│   │   │   ├── Logo + 名称 + 行业标签
│   │   │   ├── 注册信息 (时间/资本/法人)
│   │   │   └── 联系方式
│   │   ├── HealthScoreCard           // 健康指数评分卡
│   │   │   └── ScoreRing (环形评分图)
│   │   ├── TagCloud                  // 标签云 (资质/行业/区域)
│   │   └── QuickNavTabs              // 快捷导航 Tab
│   │
│   └── RightContentPanel (flex-1 可滚动)
│       ├── SectionTabBar              // 分区 Tab 导航
│       ├── DevTrendChartSection       // 发展趋势图 (折线/柱状混合)
│       ├── RiskAssessmentPanel        // 风险评估面板
│       ├── InvestPathTimeline         // 投资路径时间线
│       ├── ForeignInvestMap           // 外商投资地图
│       ├── LicenseTable               // 许可证表格
│       └── OpinionWordCloud           // 舆情词云
```

#### **Type E: 列表 / 搜索页** (约 30 页)

**适用模块**: 精准招商(大部分)、招商情报(全部)、企业监测(筛选器)、园区榜单、名企榜单

**典型页面示例**: 目标企业推荐、补链目标库、招商情报搜索、企业综合榜、垂直榜单

**组件结构**:

```
PageContainer
├── PageHeader
│
├── SearchFilterBar
│   ├── KeywordInput                // 关键词搜索
│   ├── DropdownFilters              // 下拉筛选 (产业/区域/规模/资质...)
│   ├── AdvancedToggle               // 高级筛选展开/折叠
│   └── [搜索] [重置] 按钮
│
├── ResultActionBar
│   ├── ResultCount                  // 结果计数 ("共找到 1,234 条")
│   ├── SortDropdown                 // 排序方式
│   ├── ViewToggle                   // 视图切换 (卡片/列表/表格)
│   └── ExportButton                 // 导出按钮
│
├── ResultContent (条件渲染视图模式)
│   ├── CardGridView                 // 卡片网格视图
│   │   └── EnterpriseCard × N       // 企业卡片 (头像+名称+核心指标+标签)
│   │
│   ├── ListView                     // 列表视图 (更多字段)
│   │   └── EnterpriseListItem × N
│   │
│   └── TableView                    // 表格视图 (密集数据)
│       └── el-table (name, industry, scale, location, status, actions)
│
└── PaginationBar                    // 分页组件
```

│

├── ComparisonMatrix

### 4.2 页面类型分布汇总

| 页面类型                | 大约页数 | 主要使用模块                   | 核心技术               |
| ----------------------- | -------- | ------------------------------ | ---------------------- |
| **Type A: 仪表盘/分析** | ~35      | 运行分析、企业监测、区域对标   | ECharts 密集           |
| **Type B: 地图**        | ~15      | 产业地图、区域对标、区域地图   | Maptalks/BaiduMap      |
| **Type C: 图谱/关系图** | ~12      | 产业图谱、关系链招商、企业画像 | D3 Force Graph         |
| **Type D: 画像/详情**   | ~18      | 企业监测、精准招商(评估类)     | 混合(图表+表格+信息卡) |
| **Type E: 列表/搜索**   | ~30      | 精准招商、招商情报、榜单       | el-table/el-card       |
| **Type F: 报告/对比**   | ~20      | 运行分析(报告)、对标、评价     | ECharts + 对比矩阵     |

### 4.3 通用布局壳 (MainLayout)

所有需登录后访问的页面均包裹在统一布局壳内:

```
MainLayout.vue
├── AppHeader.vue                    // 顶部导航栏 (fixed, h=56px)
│   ├── Logo + 平台名称
│   ├── 面包屑导航
│   └── 用户头像下拉菜单
│
├── AppSidebar.vue                   // 左侧菜单栏 (fixed, w=220px, 可折叠至64px)
│   └── MenuTree (递归渲染, 支持3级嵌套)
│       ├── Module 1: 产业地图 (icon)
│       │   ├── 产业链地图
│       │   ├── 供应链地图
│       │   └── 技术链地图
│       ├── Module 2: 产业图谱 (icon)
│       │   ├── ...
│       └── ... (Module 3~10)
│
└── <router-view />                  // 主内容区 (margin-left=sidebar-width)
```
