> **版本**: v1.0

## 一、功能架构

### 1.1 模块总览

| 序号 | 一级模块     | 二级模块数 | 三级功能数 | 核心特征                     |
| :--: | ------------ | :--------: | :--------: | ---------------------------- |
|  1   | **产业地图** |     3      |     9      | 地图为核心，GIS 可视化       |

|  5   | **区域对标** |     2      |     8      | 对比分析 + 热力图            |

- 产业链数量：5 条产业链

#### **模块一：产业地图**

**1.1 产业链地图**
| 功能 | 说明 |
|------|------|
| **产业资源搜索** | 用户通过关键字模糊查询、菜单列表查询以及各种条件搜索。以便于对四川省德阳市凯州新城的产业集群、产业空间和规上企业等进行查询，快速定位产业资源 |
| **产业资源筛选** | 用户在地图上按所属产业、所属行业、所属区域、所属空间、企业规模（规上企业）和是否百强等条件筛选产业资源 |
| **产业图层切换** | 用户切换显示不同的图层信息，用户可以对产业集群的热力图、企业分布图、产业空间、规上企业等图层进行切换 |
| **产业空间分布** | 基于GIS地图查看"3+1"园区平台、产业空间载体地理分布情况，并提供对产业空间信息进行查询和统计分析功能 |
| **产业集群分布** | 提供企业工商数据、行业类型数据和"3+1+1"主导产业信息的关联匹配，同时基于GIS地图，支持对全市主导产业集聚情况以热力图方式进行可视化展示，支持通过颜色深浅反映出不同区域、各种行业当地企业的分布密集程度，系统支持用户切换不同产业进行热力图查看 |

**1.2 供应链地图**
| 功能 | 说明 |
|------|------|
| **规上企业分布** | 提供对全市各区域规上企业分布情况进行GIS地图展示分析，并支持展示各区域规上企业总数、产值、增速以及规上企业基础信息、地理位置、网站、联系方式、图片、经营信息等 |
| **配套资源分布** | 用户切换至产业配套资源图层，查看全市各区域范围内的教育机构、医疗机构、银行分支、政府机构、交通站点、餐饮酒店等配套资源情况 |

**1.3 技术链地图**
| 功能 | 说明 |
|------|------|
| **产业地图操作** | 提供产业地图的缩放、浏览、视图切换、鹰眼视图、量算、查询定位等操作功能 |
| **产业地图配置** | 基于市自然资源和规划局获取的底图数据信息，提供对产业地图操作和图层等方面的配置管理，实现产业资源位置数据关联、产业资源兴趣点标注、产业资源图层管理功能 |

#### **模块三：运行分析**

**3.1 产业总览**
| 功能 | 说明 |
|------|------|
| **产业招商数据概览** | 查看园区招商数据基本概览。包括园区数量、楼栋数量、房源数量、建筑面积、可用面积、入驻企业、渠道人员数量、招商客户数量，支持查看不同园区的多维度的招商数据 |
| **载体空间数据** | 按照面积/数量查看载体空间的空间类型统计数据，按照面积/数量查看载体空间当前租售状态统计数据，按周/月维度统计园区入驻率趋势、统计园区的热门房源 |
| **渠道空间数据** | 分类统计渠道人员的数量、统计不同类型渠道带来的客户数量，统计渠道人员的变化趋势、统计渠道带来的客户变化趋势 |
| **招商管理数据** | 统计客户所处的招商阶段、按照行业统计客户数量、按照客户性质统计客户数量，按周/月统计客户变化趋势 |

**3.2 企业布局**
| 功能 | 说明 |
|------|------|
| **关联企业** | 通过互联网采集、提取产业链上下游企业的相关信息，采用规则匹配或者机器学习的方法，将企业与产业链环节进行自动匹配。并支持用户查看产业链某个环节的企业列表信息和详情信息 |
| **龙头企业** | 统计分析产业龙头企业的经营效益，包括主营业务收入、利润总额、利润支出、亏损面、资产负债率和成本费用利润率 |

**3.3 产业链整体评价**
| 功能 | 说明 |
|------|------|
| **盈利能力** | 由营业收入、营业收入增长率、总资产收益率（ROA）、净资产收益率（ROE）、行业增加值和行业增加值增长率加权得到 |
| **资本估值指数** | 由行业上市公司的市盈率、市净率、市销率以及行业上市公司的数量加权得到 |
| **投资扩张指数** | 由资本投资和生产要素投入指标来衡量，包括产业固定资产投资增速、产业企业数量、产业股权投资额、产业股权投资额增速等 |
| **议价能力指数** | 由产品价格指标来衡量，包括PPI（生产价格指数）、CPI（消费者物价指数）和大宗商品价格变动情况 |
| **议价能力指数** | 由生产景气度指标来衡量，包括产能利用率、能源消耗和能源消耗增速等。通过对盈利能力、资本估值、议价能力、投资扩张和产能利用五大评价维度的指标进行加权分析后，得到各个维度的指数分值，根据各维度分值按规则匹配生成相的产业诊断报告，提出可行性建议辅助精准决策 |

**3.4 产业链载体分析**
| 功能 | 说明 |
|------|------|
| **产业规模及预测** | 统计分析主导产业的产值和增速的趋势，并对未来三年的产值进行预测。实现动态掌握产业的整体的市场情况以及未来的发展趋势。用户可以通过切换区域和产业对产值进行分析 |
| **关联企业分析** | 对产业关联企业的注册资本和企业年龄的分布情况进行统计分析。能够掌握产业关联企业中成熟和优质企业的占比情况便于有针对性招商引资 |
| **投资热点分析** | 对关联企业对外投资情况进行分析，包括对外投资热点环节和对外投资的热点区域。统计分析产业各细分领域的投资规模和热点区域，实时掌握细分领域的投资扩张情况 |

**3.5 产业链强弱分析**
| 功能 | 说明 |
|------|------|
| **产业结构分析** | 统计分析产业结构的占比和趋势，产业结构占比主要是二级产业或细分领域的产值占比情况，产业结构趋势分析各区域劳动生产率和销售利润率的关系 |
| **产业空间分布** | 统计分析区域产值分布（雷达图展示前五位区域产值的分布情况）、区域重点企业分布（雷达图展示前五位区域重点企业的分布情况）情况，并按照产值和重点企业数量展示前十位区域的列表 |

**3.6 产业运行报告**
| 功能 | 说明 |
|------|------|
| **数据采集与整合** | 整合园区企业填报数据,政府部门数据,互联网公开数据 |
| **数据分析与可视化** | 构建科学合理的产业运行指标体系，涵盖产业规模、产业结构、产业效益、产业创新、产业环境等多个维度，运用统计分析、趋势分析、对比分析、关联分析等方法，对产业运行数据进行深入挖掘和分析，采用图表、地图、仪表盘等多种可视化形式，直观展示产业运行数据和分析结果 |
| **报告模板** | 提供多种报告模板，包括月度报告、季度报告、年度报告、专项报告等 |

**3.7 园区对比报告**
| 功能 | 说明 |
|------|------|
| **报告分析** | 园区对比报告功能旨在帮助用户对比分析不同园区的产业发展情况，为投资选址、产业布局等决策提供参考依据。该功能需支持多维度、多指标的园区对比，并提供直观的可视化展示和深入的数据分析 |
| **对比维度** | 基础信息对比，产业发展对比，经济效益对比，创新能力对比，人才资源对比，营商环境对比 |
| **对比指标** | 通用指标，特色指标 |

**3.8 企业运行报告**
| 功能 | 说明 |
|------|------|
| **数据采集与整合** | 整合园区企业填报数据,政府部门数据,互联网公开数据，第三方数据 |
| **企业运行指标体系** | 经营健康度，创新能力，人力资源，可持续发展，风险预警 |
| **报告生成与输出** | 提供标准报告，专项报告，定制报告 |

#### **模块四：产业动态**

**4.1 产业新闻热点**
| 功能 | 说明 |
|------|------|
| **产业新闻热点采集** | 产业新闻热点功能旨在为园区管理者、招商团队及入驻企业提供实时、精准的产业资讯与趋势洞察，覆盖政策动态、行业前沿、技术突破、市场变化等关键领域。通过智能采集、多维度分析和个性化推送，助力用户快速把握产业动向，辅助招商决策、企业服务及战略规划 |
| **产业新闻热点智能分析** | 政策解读：自动提取政策核心内容（如补贴金额、申报条件、截止时间），生成"政策摘要卡"。热点追踪：识别突发性事件（如"某龙头企业IPO"），自动生成事件时间轴与关联新闻聚合 |

**4.2 产业政策规划**
| 功能 | 说明 |
|------|------|
| **政策汇编** | 提供国家、市区有关产业发展规划政策、产业扶持政策、人才引进政策、住房保障政策等有关政策信息展示 |
| **政策实施监测** | 跟踪政策实施的关键节点（如资金拨付、项目验收），生成进度报告，统计政策惠及的企业数量、资金规模、就业岗位等数据 |

#### **模块五：区域对标**

**5.1 产业对比分析**
| 功能 | 说明 |
|------|------|
| **产业规模** | 企业数量、总产值、增加值、税收贡献、固定资产投资。- 单位面积产值、人均产值 |
| **产业结构** | 主导产业占比、细分行业分布（如电子信息产业中的集成电路、显示面板）。- 产业链完整性（上游原材料、中游制造、下游应用） |
| **经济效益** | 利润率、营收增长率、税收增长率、投资回报率。- 单位能耗产值、碳排放强度 |
| **创新能力** | 研发投入占比、高新技术企业数量、专利授权量、技术交易额。- 创新平台数量（如实验室、研究院） |
| **人才资源** | 从业人员总数、高层次人才占比、人才引进政策力度。- 高校与科研机构分布 |

**5.2 产业占比分析**
| 功能 | 说明 |
|------|------|
| **全局产业分布** | 支持饼图、环形图、堆叠柱状图、树状图等多维度展示各产业（如制造业、信息技术、生物医药等）的产值占比、税收贡献、企业数量占比、就业人数占比等核心指标 |
| **地图热力图** | 结合地理信息系统（GIS），在地图上分层显示不同区域的产业聚集度（如高密度工业区、新兴科技园区等），支持按产业类型筛选 |
| **动态数据看板** | 实时更新产业数据，支持用户自定义指标组合（如"产值+企业数量"联动分析） |

#### **模块六：风险预警**

**6.1 产业发展预警**
| 功能 | 说明 |
|------|------|
| **核心监测指标** | 规模指标：产值占比、企业数量占比、固定资产投资增速。效益指标：税收贡献率、利润率、单位面积产出。风险指标：产能利用率、库存周转率、应收账款周期。可持续指标：能耗强度、碳排放量、污染排放合规率。创新指标：研发投入占比、专利数量、高新技术企业占比 |
| **动态阈值设定** | 静态阈值：基于历史数据或行业标准（如"制造业产能利用率<75%"触发预警）。动态阈值：通过机器学习（如K-means聚类）划分同类区域/产业的合理区间，自动调整阈值 |
| **预警分析模型** | 异常检测模型，趋势预测模型，风险传导分析 |

**6.2 企业外迁预警**
| 功能 | 说明 |
|------|------|
| **预警指标体系** | 直接外迁信号（强关联指标）：工商变更、投资收缩、用工波动、用地退租 |
| **间接风险信号** | 经营异常、供应链变化、舆情监测、政策敏感度 |
| **风险评分模型** | 逻辑回归/Random Forest：基于历史外迁企业数据训练，输出外迁概率（0-100分） |

#### **模块七：区域地图**

#### **模块八：精准招商**（最大模块）

**8.1 产业链招商**
| 功能 | 说明 |
|------|------|
| **产业链结构** | 展示园区主导产业的上下游环节（如原材料→零部件→整机制造→服务），标注本地现有企业分布及缺失环节。支持按技术层级（基础材料、核心部件、终端产品）或应用场景（如新能源装备中的风电、氢能）分层查看 |
| **关键节点** | 标注"卡脖子"环节（如依赖进口的芯片）、高附加值环节（如研发设计）、政策敏感环节（如双高限制类） |
| **竞争力评估** | 对比全国/区域同类产业链成熟度（如成本优势、技术专利密度） |
| **目标企业智能推荐** | 企业匹配引擎，企业画像标签 |

**8.2 产业地图招商**
| 功能 | 说明 |
|------|------|
| **产业资源分布** | 分层显示不同产业的密度分布（如新能源汽车企业集群、半导体材料园区），按产值、企业数量、专利密度等指标生成热力图层，标注关键资源位置：港口、铁路枢纽、高校/科研院所、矿产分布等 |
| **产业链空间图谱** | 在地图上展示产业链上下游企业的地理位置及物流路径（如原材料供应商→制造基地→分销中心），标识产业链关键堵点（如跨省运输瓶颈、配套缺失区域） |
| **优势环节识别** | 通过产值占比、技术专利密度、市场占有率等指标，定位园区/区域的优势产业环节（如"光伏硅片制造"） |
| **延伸方向推荐** | 基于产业链图谱，推荐可向上下游延伸的高附加值环节（如"锂电池制造→电池回收"或"原材料→纳米改性材料"） |

**8.3 强延补链招商**
| 功能 | 说明 |
|------|------|
| **对标管理工具** | 与全球/全国标杆园区对比优势环节差距（如成本、效率、研发投入），生成赶超路径（技术升级、规模扩张） |
| **缺口智能诊断** | 算产业链关键环节的本地配套率（如"70%电机依赖进口"），标注断点等级（紧急/重要/一般）。识别"隐性缺口"：虽有关联企业，但技术代差超过3年 |
| **补链目标库** | 建立"可替代供应商库"，区分优先级：本地培育：技术匹配度≥60%的潜力企业，推荐孵化政策。外部招引：行业龙头或"隐形冠军"，生成定制化招商方案 |

**8.4 关系链招商**
| 功能 | 说明 |
|------|------|
| **关系图谱** | 供应链关系：企业上下游合作数据（客户/供应商清单）。投资关系：股东/子公司/对外投资网络（穿透至实际控制人）。人才流动：高管/技术骨干的跨企业任职轨迹。创新合作：联合研发、专利共同申请、产学研项目 |
| **链主企业辐射力分析** | 识别对产业链有强控制力的企业（如订单占比超30%），预测其扩张或外迁对关联企业的影响 |
| **二级关联招商** | 现有企业A，推荐其供应商的客户（B）、竞争对手的合作伙伴（C）等间接关联目标 |

**8.5 以商招商**
| 功能 | 说明 |
|------|------|
| **带动能力画像** | 量化企业招商潜力：供应链影响力（本地采购占比）、投资活跃度（对外参股数）、行业号召力（协会任职）。分析企业历史行为（如是否主动引荐合作伙伴），预测其参与招商的积极性 |
| **生态网络挖掘** | 基于链主企业的供应链（上游供应商/下游客户）、投资链（参股/控股企业）、人才链（高管履历关联企业），生成潜在招商目标清单 |
| **招商激励政策** | 引荐奖励：按落地项目投资额1‰-5‰给予现金奖励。资源倾斜：优先保障链主企业用地扩容、能源指标。对落地关联企业给予"拎包入住"优惠（如免租期延长） |

**8.6 名企榜单招商**
| 功能 | 说明 |
|------|------|
| **垂直榜单** | 按行业（如"新能源百强"）、区域（如"长三角外资企业Top50"）、企业规模（如"独角兽榜单"）细分 |
| **特色榜单** | 结合地方产业需求定制（如"德阳高端装备配套企业30强"） |
| **榜单赋能招商工具** | 输入园区产业定位（如"氢能装备"），自动匹配榜单中相关企业，按协同度排序（如技术互补性、投资匹配度） |
| **品牌联动与传播** | 与权威机构（如赛迪顾问、投中网）联合发布榜单，提升公信力。举办"榜单企业城市行"活动，定向邀请目标企业考察 |

**8.7 龙头骨干招商**
| 功能 | 说明 |
|------|------|
| **龙头骨干企业识别与评估** | 通过多维数据确定目标企业，评估其影响力和带动能力 |
| **精准招商策略生成** | 根据企业需求定制政策包，提供智能推荐 |
| **生态构建与协同** | 分析产业链缺口，推动上下游企业聚集 |

**8.8 资质企业招商**
| 功能 | 说明 |
|------|------|
| **资质企业智能识别** | 整合国家级/省级资质名录（如高企认定网、专精特新公示名单）、行业认证（ISO9001、碳足迹认证）及专利数据，构建动态企业池。支持按资质类型（技术类、绿色类、质量类）、等级（国家级/省级）多维度筛选 |
| **资质企业需求画像** | 标注企业核心诉求：研发型：实验室共建、人才引进支持；生产型：定制化厂房、绿色能源保障；市场型：区域首台套补贴、应用场景开放 |
| **资源智能匹配** | 输入企业资质类型（如"国家级绿色工厂"），推荐本地可提供的绿色金融产品、低碳园区入驻资格 |

**8.9 投行基金招商**
| 功能 | 说明 |
|------|------|
| **产业基金引导** | 设立专项基金，定向投资特定产业，推动产业链发展 |
| **项目库管理** | 收集和评估潜在投资项目，供投行和基金参考 |
| **资本对接** | 将企业与投资机构匹配，促进资本流动 |

**8.10 科研机构招商**
| 功能 | 说明 |
|------|------|
| **科研资源全景库** | 收录高校、国家重点实验室、新型研发机构等数据：硬实力：专利数量、大型仪器设备（电镜、超算中心）、国家级科研项目（973/863）；软实力：院士/杰青团队、产学研合作案例（如华为-清华联合实验室） |
| **技术转化潜力评估** | 技术价值穿透：解析专利技术链：基础研究→应用研究→产品原型→市场化路径，识别可招商环节（如中试放大需求）。产业适配度分析：匹配本地产业基础（如某柔性显示技术与本地面板企业工艺兼容性），生成技术嫁接方案 |
| **人才流动与共享机制** | 建立科研机构专家在企业兼职的合规通道，标注领域（微电子、生物医药）、可服务时长（月度/季度） |

**8.11 链主适配招商**
| 功能 | 说明 |
|------|------|
| **链主画像与需求解析** | 多维标签体系：供应链控制力：上游供应商集中度、下游客户分布、关键原材料自给率。技术话语权：核心专利数量、行业标准参与度、研发投入占比。扩张动态：新基地选址偏好（如"成渝地区"）、技术合作需求（如"氢能储运技术"）。需求智能提取：解析链主企业公告、招标文件、技术白皮书，提取关键词（如"本地化采购""联合研发"） |
| **产业链缺口诊断** | 计算链主企业本地采购额占比，识别低于行业平均的环节（如"电机供应商本地化率仅30%"）。标注高度依赖进口或单一供应商的"卡脖子"环节（如"高端轴承100%进口"）。可视化产业链各环节健康度 |
| **适配企业智能匹配** | 基于链主需求与目标企业标签生成适配评分 |

**8.12 园区招商**
| 功能 | 说明 |
|------|------|
| **园区资源全景展示** | 核心优势可视化：地理禀赋：标注交通枢纽（机场、港口、高铁）、物流半径、周边产业配套。产业基础：展示主导产业占比（如装备制造占园区产值60%）、龙头企业分布。政策红利：税收优惠、人才补贴、用地保障等政策分类标签化 |
| **智能企业匹配引擎** | 企业输入需求：行业类型（如生物医药）、面积需求（5000㎡±10%）、员工规模（200人+）。智能匹配园区可用地块/厂房，按适配度排序（区位>成本>政策） |
| **产业链协同招商** | 产业链缺口诊断：分析园区现有产业链薄弱环节（如某环节本地配套率<40%），推荐补链企业清单。链主企业牵引：为园区内链主企业（如东电）生成"供应商引入奖励包"，激励其推荐上下游伙伴 |

**9.5 资本招商情报**

**9.6 招商风险监测**

| **风险分类体系** | 政策风险：产业限制政策变动（如"双高"行业限批）、补贴退坡预警。市场风险：行业周期波动（如光伏产能过剩）、技术替代威胁（如氢能替代锂电池）。企业风险：经营异常（连续3个月纳税下降30%+）、信用评级下调（如穆迪降级）。履约风险：投资进度延迟（如厂房建设滞后6个月）、配套承诺未兑现（如链主未完成本地采购量） |

**9.8 我的招商情报**

#### **模块十：企业监测**

**10.1 企业总览**
| 功能 | 说明 |
|------|------|
| **基础信息总览** | 企业基础档案：企业名称、注册时间、行业分类（清洁能源装备/先进材料/电子信息）、股权结构（本地/外资占比）。德阳示例：标注东电、二重等链主企业及其控股子公司 |
| **经济贡献分析** | 年度总产值,清洁能源装备产值,税收贡献Top3企业 |
| **动态排名** | 按纳税/就业/研发投入生成企业榜单 |

**10.2 经营健康度监测**
| 功能 | 说明 |
|------|------|
| **关键指标看板** | 关键指标看板，纳税波动率，用电量/产能比，员工稳定性。动态评分 |
| **产业链协同监测** | 供应链韧性分析：本地配套率实时计算（本地供应商订单额/总采购额），标注低于行业均值（如<60%）的环节 |
| **可持续发展监测** | 绿色生产合规性：单位产值碳排放，危废处置合规率。绿电使用追踪：园区绿电覆盖率（当前60%），标注未达标企业清单及绿电配额分配建议 |

**10.3 企业榜单**
| 功能 | 说明 |
|------|------|
| **多源榜单动态整合** | 聚焦榜单数据的动态追踪、风险预警与价值联动，通过实时校验、趋势分析与策略适配，保障榜单的决策参考价值，助力精准招商 |
| **榜单波动预警体系** | 异常波动检测：横向波动：同一企业在多榜单排名差异≥20位（如A榜单第10，B榜单第50）；纵向波动：季度排名下降≥15位或专利数突增 |

**10.4 企业筛选器**
| 功能 | 说明 |
|------|------|
| **基础信息筛选** | 行业分类：支持多级行业标签（如"新能源→储能→电池回收"）。企业规模：按营收（1亿+/10亿+）、员工数（100人+/500人+）、用地面积（50亩+/200亩+）分层。地理围栏：地图圈选（如成渝经济圈）、物流半径（500公里内覆盖港口/机场） |
| **经营与资质筛选** | 经营健康度:纳税稳定性,研发投入强度 |
| **产业链关联筛选** | 输入本地链主企业（如东电集团），筛选其一级供应商/客户（订单占比≥10%）。技术协同：专利关键词匹配（如"氢能储运""高温合金"） |
| **风险与合规筛选** | 风险排除：黑名单过滤（环保处罚、重大诉讼）、财务风险（资产负债率≥70%）。合规准入：强制匹配园区准入标准（如"单位产值能耗≤0.5吨标煤/万元"） |

**10.5 企业画像**
| 功能 | 说明 |
|------|------|
| **企业健康指数** | 优秀，良好，预警，高危 |
| **多维度画像** | 工商数据：股权结构（实控人穿透）、分支机构、对外投资（子公司/参股）。资质标签：高企、专精特新、绿色工厂等认证状态及有效期。地理坐标：总部/工厂位置、物流辐射范围（500公里覆盖港口/机场） |
| **产业链关联** | 供应链图谱：核心供应商（采购额Top5）、关键客户（销售额Top5）、技术合作方。控制力指数：基于订单占比、专利引用数计算企业产业链话语权（0-100分） |

**10.6 企业综合评价**
