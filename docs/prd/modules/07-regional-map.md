#### **模块七：区域地图**

**7.1 园区榜单**
| 功能 | 说明 |
|------|------|
| **榜单指标体系** | 经济效益维度（基础权重40%）：年度纳税额，单位面积产出，营收增长率，固定资产投资。创新活力维度（权重30%）：研发投入占比，知识产权密度，技术合同成交额。社会贡献维度（权重20%）：就业带动指数，低碳合规率，公益投入。成长潜力维度（权重10%）：融资能力，市场扩张速度，政策契合度 |
| **企业综合榜** | 全维度加权排名（Top 100） |
| **产业链协同榜** | 按产业链环节排名（如"核心零部件企业Top5"） |
