# 模块十：企业监测

> [返回目录](../index.md)

#### **模块十：企业监测**

**10.1 企业总览**
| 功能 | 说明 |
|------|------|
| **基础信息总览** | 企业基础档案：企业名称、注册时间、行业分类（清洁能源装备/先进材料/电子信息）、股权结构（本地/外资占比）。德阳示例：标注东电、二重等链主企业及其控股子公司 |
| **经济贡献分析** | 年度总产值,清洁能源装备产值,税收贡献Top3企业 |
| **动态排名** | 按纳税/就业/研发投入生成企业榜单 |

**10.2 经营健康度监测**
| 功能 | 说明 |
|------|------|
| **关键指标看板** | 关键指标看板，纳税波动率，用电量/产能比，员工稳定性。动态评分 |
| **产业链协同监测** | 供应链韧性分析：本地配套率实时计算（本地供应商订单额/总采购额），标注低于行业均值（如<60%）的环节 |
| **可持续发展监测** | 绿色生产合规性：单位产值碳排放，危废处置合规率。绿电使用追踪：园区绿电覆盖率（当前60%），标注未达标企业清单及绿电配额分配建议 |

**10.3 企业榜单**
| 功能 | 说明 |
|------|------|
| **多源榜单动态整合** | 聚焦榜单数据的动态追踪、风险预警与价值联动，通过实时校验、趋势分析与策略适配，保障榜单的决策参考价值，助力精准招商 |
| **榜单波动预警体系** | 异常波动检测：横向波动：同一企业在多榜单排名差异≥20位（如A榜单第10，B榜单第50）；纵向波动：季度排名下降≥15位或专利数突增 |

**10.4 企业筛选器**
| 功能 | 说明 |
|------|------|
| **基础信息筛选** | 行业分类：支持多级行业标签（如"新能源→储能→电池回收"）。企业规模：按营收（1亿+/10亿+）、员工数（100人+/500人+）、用地面积（50亩+/200亩+）分层。地理围栏：地图圈选（如成渝经济圈）、物流半径（500公里内覆盖港口/机场） |
| **经营与资质筛选** | 经营健康度:纳税稳定性,研发投入强度 |
| **产业链关联筛选** | 输入本地链主企业（如东电集团），筛选其一级供应商/客户（订单占比≥10%）。技术协同：专利关键词匹配（如"氢能储运""高温合金"） |
| **风险与合规筛选** | 风险排除：黑名单过滤（环保处罚、重大诉讼）、财务风险（资产负债率≥70%）。合规准入：强制匹配园区准入标准（如"单位产值能耗≤0.5吨标煤/万元"） |

**10.5 企业画像**
| 功能 | 说明 |
|------|------|
| **企业健康指数** | 优秀，良好，预警，高危 |
| **多维度画像** | 工商数据：股权结构（实控人穿透）、分支机构、对外投资（子公司/参股）。资质标签：高企、专精特新、绿色工厂等认证状态及有效期。地理坐标：总部/工厂位置、物流辐射范围（500公里覆盖港口/机场） |
| **产业链关联** | 供应链图谱：核心供应商（采购额Top5）、关键客户（销售额Top5）、技术合作方。控制力指数：基于订单占比、专利引用数计算企业产业链话语权（0-100分） |

**10.6 企业综合评价**
| 功能 | 说明 |
|------|------|
| **五维评价模型** | 经营质量:营收增速、净利率、现金流/负债比。创新能力:有效专利数、研发投入占比、技术转化率。产业协同:本地采购率、链主企业订单占比、专利引用数。风险韧性:ESG评分、环保处罚记录、供应链备份指数。政策适配:资质匹配度、政策申报成功率、补贴依赖度 |
