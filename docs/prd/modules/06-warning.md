# 模块六：风险预警

#### **模块六：风险预警**

**6.1 产业发展预警**
| 功能 | 说明 |
|------|------|
| **核心监测指标** | 规模指标：产值占比、企业数量占比、固定资产投资增速。效益指标：税收贡献率、利润率、单位面积产出。风险指标：产能利用率、库存周转率、应收账款周期。可持续指标：能耗强度、碳排放量、污染排放合规率。创新指标：研发投入占比、专利数量、高新技术企业占比 |
| **动态阈值设定** | 静态阈值：基于历史数据或行业标准（如"制造业产能利用率<75%"触发预警）。动态阈值：通过机器学习（如K-means聚类）划分同类区域/产业的合理区间，自动调整阈值 |
| **预警分析模型** | 异常检测模型，趋势预测模型，风险传导分析 |

**6.2 企业外迁预警**
| 功能 | 说明 |
|------|------|
| **预警指标体系** | 直接外迁信号（强关联指标）：工商变更、投资收缩、用工波动、用地退租 |
| **间接风险信号** | 经营异常、供应链变化、舆情监测、政策敏感度 |
| **风险评分模型** | 逻辑回归/Random Forest：基于历史外迁企业数据训练，输出外迁概率（0-100分） |
