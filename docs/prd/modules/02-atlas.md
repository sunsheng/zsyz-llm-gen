# 模块二：产业图谱

> [返回目录](../index.md)

#### **模块二：产业图谱**

**2.1 全国图谱**
| 功能 | 说明 |
|------|------|
| **产业链图谱构建** | 提供全国高端装备制造（汽车汽配）、前沿材料（玻纤及复合材料、新能源材料、碳纤维及复合材料）、数字经济（5G、集成电路、先进计算、智能安防）、时尚产业产业链上、中、下游环节梳理和产业链图谱构建 |
| **产业链关联企业** | 通过互联网采集、提取全国产业链上下游企业的相关信息，采用规则匹配或者机器学习的方法，将企业与产业链环节进行自动匹配。并支持用户查看产业链某个环节的企业列表信息和详情信息 |
| **产业链关键环节识别** | 基于全国高端装备制造（汽车汽配）、新材料、数字经济、时尚产业、现代服务业主导产业链条上面所形成的数据集，从产业产值、头部企业数量、市场占有率、进出口等方面评估和识别四川省德阳市凯州新城产业和企业在整个产业链条中的主导环节和薄弱环节 |
| **产业链图谱管理** | 为用户提供全国产业链图谱配置、产业链图谱增删改查、产业链图谱关联企业管理、图谱定义、图谱创建等功能 |
