# 产业招商平台 — 产品需求文档 (PRD)

> **版本**: v1.0  
> **日期**: 2026-05-12  
> **技术栈**: Vue 3 + Vite + Element Plus  
> **功能依据**: docs/九、产业招商平台.xlsx

---

## 一、功能架构

### 1.1 模块总览

| 序号 | 一级模块     | 二级模块数 | 三级功能数 | 核心特征                     |
| :--: | ------------ | :--------: | :--------: | ---------------------------- |
|  1   | **产业地图** |     3      |     9      | 地图为核心，GIS 可视化       |
|  2   | **产业图谱** |     2      |     8      | 力导向关系图 + 企业关联      |
|  3   | **运行分析** |     8      |     22     | 最重图表模块（ECharts 密集） |
|  4   | **产业动态** |     2      |     4      | 新闻 + 政策时间线            |
|  5   | **区域对标** |     2      |     8      | 对比分析 + 热力图            |
|  6   | **风险预警** |     2      |     6      | 预警仪表盘 + 评分模型        |
|  7   | **区域地图** |     1      |     3      | 园区榜单排名                 |
|  8   | **精准招商** |     12     |   **39**   | 最大模块，12 种招商模式      |
|  9   | **招商情报** |     8      |     9      | 多源情报聚合 + 全文搜索      |
|  10  | **企业监测** |     6      |     17     | 企业画像 + 筛选器 + 五维评价 |

**数据支撑模块**（无独立页面，为其他模块提供数据）：

- 产业图谱数据（单条链）：精选产业链图谱、企业产业标签
- 企业图谱数据（单条链）：工商数据、供应链图谱、园区企业库、资质企业库、私募创投库、新闻舆情、政策规划、招中标库
- 决策评价数据：企业评价、产业评价
- 产业链数量：5 条产业链

### 1.2 完整功能层级树

---

#### **模块一：产业地图**

**1.1 产业链地图**
| 功能 | 说明 |
|------|------|
| **产业资源搜索** | 关键字模糊查询 + 菜单列表查询 + 多条件组合搜索，快速定位产业集群/产业空间/规上企业 |
| **产业资源筛选** | 在地图上按所属产业、行业、区域、空间、企业规模（规上）、是否百强等条件筛选 |
| **产业图层切换** | 切换显示不同图层：产业集群热力图、企业分布图、产业空间、规上企业等 |
| **产业空间分布** | 基于 GIS 地图查看"3+1"园区平台、产业空间载体地理分布，支持查询和统计分析 |
| **产业集群分布** | 关联企业工商数据+行业类型+"3+1+1"主导产业信息，以热力图方式可视化展示，支持切换不同产业查看 |

**1.2 供应链地图**
| 功能 | 说明 |
|------|------|
| **规上企业分布** | 各区域规上企业 GIS 展示分析，含各区域总数、产值、增速及基础信息/位置/网站/联系方式/图片/经营信息 |
| **配套资源分布** | 切换至配套资源图层，查看教育机构/医疗机构/银行分支/政府机构/交通站点/餐饮酒店等 |

**1.3 技术链地图**
| 功能 | 说明 |
|------|------|
| **产业地图操作** | 缩放、浏览、视图切换、鹰眼视图、量算、查询定位等操作功能 |
| **产业地图配置** | 基于底图数据，提供地图操作和图层配置管理，实现位置数据关联、兴趣点标注、图层管理 |

#### **模块二：产业图谱**

**2.1 全国图谱**
| 功能 | 说明 |
|------|------|
| **产业链图谱构建** | 高端装备制造（汽车汽配）、前沿材料（玻纤/新能源/碳纤维）、数字经济（5G/集成电路/先进计算/智能安防）、时尚产业的上中下游梳理和图谱构建 |
| **产业链关联企业** | 互联网采集提取全国上下游企业，规则匹配/机器学习自动匹配，查看某环节企业列表和详情 |
| **产业链关键环节识别** | 从产值、头部企业数、市场占有率、进出口等维度评估识别桐乡市产业在链条中的主导环节和薄弱环节 |
| **产业链图谱管理** | 图谱配置、增删改查、关联企业管理、图谱定义、创建等功能 |

**2.2 区域图谱**
| 功能 | 说明 |
|------|------|
| **产业链图谱构建** | 园区层面的高端装备制造/前沿材料/数字经济/时尚产业上中下游梳理和图谱构建 |
| **产业链关联企业** | 园区层面企业采集匹配，查看某环节企业列表和详情 |
| **产业链关键环节识别** | 园区层面评估识别主导环节和薄弱环节 |
| **产业链图谱管理** | 园区层面图谱配置和管理功能 |

#### **模块三：运行分析**

**3.1 产业总览**
| 功能 | 说明 |
|------|------|
| **产业招商数据概览** | 园区数量、楼栋数量、房源数量、建筑面积、可用面积、入驻企业、渠道人员、招商客户数量，支持多园区多维度查看 |
| **载体空间数据** | 按面积/数量查看空间类型统计，按面积/数量查看租售状态统计，周/月入驻率趋势，热门房源统计 |
| **渠道空间数据** | 分类统计渠道人员数量，不同类型渠道客户数量，渠道人员变化趋势，客户变化趋势 |
| **招商管理数据** | 统计客户所处招商阶段，行业/性质分类统计，周/月客户变化趋势 |

**3.2 企业布局**
| 功能 | 说明 |
|------|------|
| **关联企业** | 采集提取上下游企业，自动匹配产业链环节，查看企业列表和详情 |
| **龙头企业** | 统计分析龙头企业经营效益：主营业务收入、利润总额、利润支出、亏损面、资产负债率、成本费用利润率 |

**3.3 产业链整体评价**
| 功能 | 说明 |
|------|------|
| **盈利能力** | 营业收入、增长率、ROA、ROE、行业增加值及其增长率加权得到 |
| **资本估值指数** | 行业上市公司市盈率、市净率、市销率及上市公司数量加权得到 |
| **投资扩张指数** | 固定资产投资增速、企业数量、股权投资额及增速等衡量 |
| **议价能力指数(价格)** | PPI、CPI、大宗商品价格变动情况衡量 |
| **议价能力指数(产能)** | 产能利用率、能源消耗及增速等衡量。五大维度加权分析后生成诊断报告和可行性建议 |

**3.4 产业链载体分析**
| 功能 | 说明 |
|------|------|
| **产业规模及预测** | 主导产业产值和增速趋势，未来三年产值预测，支持切换区域和产业 |
| **关联企业分析** | 注册资本和企业年龄分布统计，掌握成熟优质企业占比便于精准招商 |
| **投资热点分析** | 对外投资热点环节和热点区域分析，细分领域投资规模和热点区域实时掌握 |

**3.5 产业链强弱分析**
| 功能 | 说明 |
|------|------|
| **产业结构分析** | 二级产业/细分领域产值占比，劳动生产率和销售利润率关系趋势 |
| **产业空间分布** | 区域产值分布（雷达图前五位）、重点企业分布（雷达图前五位），前十位列表展示 |

**3.6 产业运行报告**
| 功能 | 说明 |
|------|------|
| **数据采集与整合** | 整合园区企业填报、政府部门数据、互联网公开数据 |
| **数据分析与可视化** | 构建科学指标体系（产业规模/结构/效益/创新/环境等多维度），采用图表/地图/仪表盘等多种形式展示 |
| **报告模板** | 月度报告、季度报告、年度报告、专项报告等多种模板 |

**3.7 园区对比报告**
| 功能 | 说明 |
|------|------|
| **报告分析** | 多维度多指标园区对比，直观可视化展示和深入数据分析 |
| **对比维度** | 基础信息对比、产业发展对比、经济效益对比、创新能力对比、人才资源对比、营商环境对比（6项） |
| **对比指标** | 通用指标、特色指标 |

**3.8 企业运行报告**
| 功能 | 说明 |
|------|------|
| **数据采集与整合** | 整合园区企业填报、政府部门数据、互联网公开数据、第三方数据 |
| **企业运行指标体系** | 经营健康度、创新能力、人力资源、可持续发展、风险预警 |
| **报告生成与输出** | 标准报告、专项报告、定制报告 |

#### **模块四：产业动态**

**4.1 产业新闻热点**
| 功能 | 说明 |
|------|------|
| **产业新闻热点采集** | 覆盖政策动态、行业前沿、技术突破、市场变化等领域，智能采集+多维度分析+个性化推送，辅助招商决策/企业服务/战略规划 |

| 功能 | 说明 |
|------|------|

| **政策实施监测** | 跟踪实施关键节点（资金拨付/项目验收），生成进度报告，统计惠及企业数/资金规模/就业岗位等 |

#### **模块五：区域对标**

**5.1 产业对比分析**
| 功能 | 说明 |
|------|------|
| **产业规模** | 企业数量、总产值、增加值、税收贡献、固定资产投资；单位面积产值、人均产值 |
| **产业结构** | 主导产业占比、细分行业分布、产业链完整性（上游原材料/中游制造/下游应用） |
| **经济效益** | 利润率、营收增长率、税收增长率、投资回报率；单位能耗产值、碳排放强度 |
| **创新能力** | 研发投入占比、高新技术企业数量、专利授权量、技术交易额；创新平台数量 |
| **人才资源** | 从业人员总数、高层次人才占比、人才引进政策力度；高校与科研机构分布 |

**5.2 产业占比分析**
| 功能 | 说明 |
|------|------|
| **全局产业分布** | 饼图/环形图/堆叠柱状图/树状图等多维度展示各产业产值占比/税收贡献/企业数量/就业人数等核心指标 |
| **地图热力图** | 结合 GIS 在地图上分层显示不同区域产业聚集度，支持按产业类型筛选 |
| **动态数据看板** | 实时更新产业数据，支持用户自定义指标组合联动分析 |

#### **模块六：风险预警**

**6.1 产业发展预警**
| 功能 | 说明 |
|------|------|
| **核心监测指标** | 规模指标（产值/企业数/固投增速）、效益指标（税收/利润率/单位产出）、风险指标（产能利用率/库存周转率/应收账款）、可持续指标（能耗强度/碳排放/合规率）、创新指标（研发占比/专利/高企占比） |
| **动态阈值设定** | 静态阈值（基于历史或行业标准触发预警）、动态阈值（K-means 聚类划分合理区间自动调整） |
| **预警分析模型** | 异常检测模型、趋势预测模型、风险传导分析 |

**6.2 企业外迁预警**
| 功能 | 说明 |
|------|------|
| **预警指标体系** | 直接外迁信号（强关联）：工商变更、投资收缩、用工波动、用地退租 |
| **间接风险信号** | 经营异常、供应链变化、舆情监测、政策敏感度 |
| **风险评分模型** | 逻辑回归/Random Forest 基于历史外迁企业训练，输出外迁概率（0-100分） |

#### **模块七：区域地图**

#### **模块八：精准招商**（最大模块）

**8.1 产业链招商**
| 功能 | 说明 |
|------|------|
| **产业链结构** | 展示主导产业上下游环节（原材料→零部件→整机制造→服务），标注本地现有企业和缺失环节，支持技术层级或应用场景分层查看 |
| **关键节点** | 标注"卡脖子"环节（依赖进口芯片）、高附加值环节（研发设计）、政策敏感环节（双高限制类） |
| **竞争力评估** | 对比全国/区域同类产业链成熟度（成本优势、技术专利密度） |
| **目标企业智能推荐** | 企业匹配引擎、企业画像标签 |

**8.2 产业地图招商**
| 功能 | 说明 |
|------|------|
| **产业资源分布** | 分层显示不同产业密度分布（新能源汽车集群/半导体园区），按产值/企业数量/专利密度等生成热力图层，标注关键资源位置 |
| **产业链空间图谱** | 在地图上展示上下游企业地理位置及物流路径，标识关键堵点（跨省运输瓶颈/配套缺失） |
| **优势环节识别** | 通过产值占比/技术专利密度/市场占有率定位优势环节 |
| **延伸方向推荐** | 基于图谱推荐可向上下游延伸的高附加值环节 |

**8.3 强延补链招商**
| 功能 | 说明 |
|------|------|
| **对标管理工具** | 与全球/全国标杆园区对比优势环节差距，生成赶超路径 |
| **缺口智能诊断** | 计算关键环节本地配套率，标注断点等级（紧急/重要/一般），识别"隐性缺口"（技术代差超3年） |
| **补链目标库** | 建立"可替代供应商库"，区分本地培育（匹配度≥60%潜力企业推荐孵化）和外部招引（龙头/隐形冠军生成定制方案） |

**8.4 关系链招商**
| 功能 | 说明 |
|------|------|
| **关系图谱** | 供应链关系（客户/供应商清单）、投资关系（股东/子公司/对外投资穿透至实控人）、人才流动（高管跨企任职轨迹）、创新合作（联合研发/专利共申/产学研项目） |
| **链主企业辐射力分析** | 识别强控制力企业（订单占比超30%），预测扩张/外迁对关联企业影响 |
| **二级关联招商** | 推荐供应商的客户（B）、竞争对手合作伙伴（C）等间接关联目标 |

**8.5 以商招商**
| 功能 | 说明 |
|------|------|
| **带动能力画像** | 量化招商潜力：供应链影响力（本地采购占比）、投资活跃度（对外参股数）、行业号召力（协会任职），分析历史行为预测参与积极性 |
| **生态网络挖掘** | 基于链主供应链/投资链/人才链生成潜在招商目标清单 |
| **招商激励政策** | 引荐奖励（落地投资额1‰-5‰现金）、资源倾斜（用地扩容/能源优先）、关联企业"拎包入住"优惠（免租期延长） |

**8.6 名企榜单招商**
| 功能 | 说明 |
|------|------|
| **垂直榜单** | 按行业（新能源百强）、区域（长三角外资Top50）、企业规模（独角兽榜单）细分 |
| **特色榜单** | 结合地方需求定制（如高端装备配套企业30强） |
| **榜单赋能招商工具** | 输入产业定位自动匹配榜单相关企业，按协同度排序 |
| **品牌联动与传播** | 与权威机构联合发布榜单，举办"榜单企业城市行"活动定向邀请考察 |

**8.7 龙头骨干招商**
| 功能 | 说明 |
|------|------|
| **龙头骨干企业识别与评估** | 多维数据确定目标企业，评估影响力和带动能力 |
| **精准招商策略生成** | 根据企业需求定制政策包，智能推荐 |
| **生态构建与协同** | 分析产业链缺口，推动上下游聚集 |

**8.8 资质企业招商**
| 功能 | 说明 |
|------|------|

### 4.2 页面类型分布汇总

| 页面类型                | 大约页数 | 主要使用模块                   | 核心技术               |
| ----------------------- | -------- | ------------------------------ | ---------------------- |
| **Type A: 仪表盘/分析** | ~35      | 运行分析、企业监测、区域对标   | ECharts 密集           |
| **Type B: 地图**        | ~15      | 产业地图、区域对标、区域地图   | Maptalks/BaiduMap      |
| **Type C: 图谱/关系图** | ~12      | 产业图谱、关系链招商、企业画像 | D3 Force Graph         |
| **Type D: 画像/详情**   | ~18      | 企业监测、精准招商(评估类)     | 混合(图表+表格+信息卡) |
| **Type E: 列表/搜索**   | ~30      | 精准招商、招商情报、榜单       | el-table/el-card       |
| **Type F: 报告/对比**   | ~20      | 运行分析(报告)、对标、评价     | ECharts + 对比矩阵     |
